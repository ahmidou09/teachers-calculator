# Teachers' Calculator

A responsive React calculator for teachers to quickly sum fractional grade values (0.25â€“5.00). Clicking a button automatically adds that value to the running expression. Includes undo, reset, and copy-to-clipboard.

## Features

- Buttons for only fractional values: 0.25, 0.5, ..., 5.0
- Auto-add (no `+` button shown)
- Undo last entry, reset all, copy total
- Mobile-first responsive UI with Tailwind CSS
